Welcome to the Diablo II: Resurrected Data Guide!

The Data Guide is an html-format documentation of all-things data in Diablo II: Resurrected. It is used by both developers and community modders.

# Authoring

## Reference Links

In certain areas, like a file's "overview" or a field's "description", a reference link may be placed. This is a shorthand for creating an html link to another page or element. All links are opened in a new tab. The format is as follows:

`$!page!$` - Links to another page.

`$!page#element!$` - Links to a specific element in another page.

`$!#element!$` - Links to a specific element in the current page.
